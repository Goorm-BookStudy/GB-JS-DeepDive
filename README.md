### 내용
```
매일 1장(1챕터)씩 읽고 정리
- 구글 시트의 체크 필수
매주 금요일 오후 8시 회의
- 그 주에 읽은 부분을 랜덤으로 2명을 선정해 발표를 진행합니다.
- 발표에 대해서 질의응답 시간을 가지고 회의를 종료합니다.
```
